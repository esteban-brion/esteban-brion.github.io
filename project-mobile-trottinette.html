Interface Graphique pour Application de Déverrouillage de Trottinette

Ce projet consiste en la création d'une interface graphique et d'un serveur pour une application de déverrouillage de trottinette. Il permet aux utilisateurs de trouver des stations de trottinettes, de gérer leurs locations, et de déverrouiller les trottinettes via l'application mobile.

Technologies utilisées : Swift, Node.js, Firebase Firestore, Plan Maps API

Rôle : Développeur Principal

Contributions :

  • Développement de l'interface utilisateur en Swift pour iOS
  • Implémentation du back-end avec Node.js pour gérer les requêtes serveur
  • Utilisation de Firebase Firestore pour la base de données en temps réel
  • Intégration de l'API Plan Maps pour afficher les stations sur une carte
  • Développement de fonctionnalités de localisation et de gestion des locations en temps réel
  • Apprentissage autodidacte de Swift pour le développement de l'application
